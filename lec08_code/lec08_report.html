Discussion¶

決定木から花弁の特徴量が分類に適していると予測できたが、実際の特徴量の重要値からも花弁が分類に適していることが分かる。setosaは花弁がの幅と長さが短いがガクの長さは個体ごとに大きく違う。virginicaは花弁が大きくガクに大きな特徴はない。versicolorは花弁がsetosaより大きいがvirginicaより小さい。 setosaは花弁の大きさで高い純度で分類できるが、versicolorとvirginicaは大まかに分類はできるものの例外が混ざっていることがある。
